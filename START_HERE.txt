❓ COMMON QUESTIONS

Q: Do I have to do all 3 commands?
A: Yes! 1) Initialize cache 2) Start server 3) Open browser

Q: Can I run the Flask server without initializing?
A: No. Initialize first to create the cache files.

Q: What if I close the browser and reopen?
A: Cache persists! Just open browser again, it loads instantly.

Q: What if I need fresh data?
A: Click "REFRESH" button. It fetches new data (1 credit) and updates cache.

Q: Can I use this with other APIs?
A: Yes! The cache_manager.py is generic. Works with any JSON data.

Q: Does the cache auto-update?
A: No. It stays the same until you click REFRESH.

Q: How long does cache last?
A: Forever! Until you manually click REFRESH or clear it.

Q: Can I see the cache file?
A: Yes! Check: espn_live_scraper/cache/ipl_2026_matches.json

Q: How do I clear the cache?
A: Delete the espn_live_scraper/cache/ folder, or run:
   python -c "from espn_live_scraper.cache_manager import clear_cache; clear_cache()"
